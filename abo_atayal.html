泰雅族

  • 族群
  • 音樂人
泰雅族音樂常使用的音有 re、mi、sol、la。泰雅族的傳統歌謠不多，現存的歌謠大多為日據時期後的改編或創作的作品。
泰雅族的音樂可分成兩個部份來討論：
  • 1. 民歌部份：
    泰雅族的民歌音樂組織結構中，基本音組織有兩種 : 其一是三音組織，在小三度的上或下方加上大二度音 ; 另外一種是四音組織，就是在小三度音的上下加上大二度音，也就是說部份泰雅民歌的音組織是以三個主要特定音作為其中心音，而另一部份是以四個特定音為其中心音的，這兩種音組織結構都還不成為構成一較複雜音階的條件。
    目前泰雅族民歌的演唱方式屬單音旋律的唱法，形式有對唱、領唱與對唱，及二聲部或三聲部的卡農等。泰雅的住地很廣，其聚落分散各地，因此泰雅族很少有合唱音樂，甚至可以說是沒有，獨唱和對唱便成了最主要的歌唱方式。
    泰雅族有兩個亞群： 一為泰雅亞群，其歌曲以三音組織及獨唱為主 ; 一為賽德克亞群，其歌曲則以四音組織為主，有合唱音樂，變化較多而顯得更為豐富。
  • 2. 樂器部份：
    泰雅族傳統樂器的材質大都是就地取材；由於生活型態與社會組織的因素，樂器製作過程簡易不繁複，在構造與型式上亦如其生活器物一樣簡單、樸實，甚至是織布機的零件「 hon-ngu 」也被利用作為製造聲響的樂器，因此沒有像排灣、魯凱或達悟族的細緻雕飾與華麗綴飾；這現象也正可以作為支持、說明泰雅文化乃人文生態與自然環境平衡發展足夠成為典型範例的一項理由。
    過去曾使用過縱笛與木琴，現今較常見的是口簧琴。口簧琴有其特殊用法，用在傳話、舞蹈伴奏及吹奏音樂等用途。世界上有許多民族以鼓聲作為傳訊工具，因擊鼓技巧的複雜化，使得將鼓作為音樂上純粹的樂器時，亦加強了其表現力而豐富了音樂本身的表達。但泰雅族的口簧琴，則因樂器本身的構造的關係，其傳話功能有空間的限制，更由於樂器的音量，使得口簧琴在泰雅族的社會中特別顯出一種迷人的風采，而成為泰雅族戀人們互相談心的工具。

歌謠類型

  • 口簧琴

    口簧琴在泰雅族的音樂中一直有其顯著的地位，除了傳話舞蹈伴奏的用途外，純粹的樂器演奏也常被泰雅族人所應用著，演奏形態有獨奏亦有合奏。合奏時常用不同簧片數目的口簧琴一起合奏，因為各個簧的音律不同，再加上演奏者各有不同的想法，因而形成了一種異音的性格。
    口簧琴的簧數是由一到五簧皆有，四簧以上的口簧琴是只泰雅族才有的。一般多簧的口簧琴，琴台本身一樣是竹製的，而簧片的材質多用金屬 ( 銅片或鐵片 ) ，其音階與歌謠的音階是一致的。 口簧琴的吹奏法是先用左手的姆指和食指繫住口簧琴接頭相反的一端 ( 也有人將繩子捲在左手手指 ) ，將嘴靠在口簧琴的表面，接近簧的一端，用右手拉相反一方的繩子，簧即振動，同時在口腔內輕輕有規律地呼氣和吸氣，而將口腔當成共鳴箱，使其產生共鳴。音的強弱可以由右手拉動的繩子來控制。

  • 傳統舞蹈之歌

    這是一首屬於泰雅族未婚女孩子們集體所跳的傳統舞（luliz biyau）中的舞蹈歌曲。領舞的一定是一位舞技精湛的已婚婦女。首先一定由她先領唱一句，其它未婚婦女再覆頌一次。由於每次覆頌的部份都在領唱者尚未唱完之前就已進入，於是形成了非常優美、規律的輪唱（或稱卡農）形式。這種唱法是西賽德克亞群（霧社附近）獨有的歌唱方式，位在親愛村的萬大群（屬澤敖列群）因與霧社群比鄰而居，自然也學會了這種唱法。每次當領舞的人唱到「ka le bi e nu」時，就是要所有的舞者注意要換舞步了。

  • 快樂聚會歌

    這首歌和上一首傳統舞之歌，基本上是相同性質的歌舞，也都是在集體聚會的場合，未婚的女子們快樂的一起共舞，舞出最美的舞姿，唱出最亮麗的歌聲以吸引未婚異性的注意。

  • 二部舞歌輪唱

    這首二部舞歌輪唱不只是在婚禮中，就是在高興、快樂、歡慶時也可以唱。通常是由女子們圍成圓圈，一面歌唱一面跳舞。這是一首前面加上了二段歌頭似的引子，接下來是一段很精彩的二部卡農，第二部很忠實的模仿第一聲部。這本來是西賽德克亞群的歌唱形式，但是親愛、新生、互助、清流一帶泰雅族人也都會唱這種卡農歌謠。

  • 祖訓之歌

    詞意：我為今日特別來感謝，讓我來細心的吟唱。起先我要說的是，有關先祖Yaboh，他是源自於碩大的發源地，（因此地人口愈來愈多，族人已經無居住之處），因此，Yaboh已經囑咐他的兒女們，順著河流處，還是先祖Kmyaboh所吩咐的。源自於梨山來的Yuraw Pasary也同樣來此地（河流處）會合。另外，你們（指後代）無論如何不可忘記兩件事： （1）當你們組成家族後，必須要認真工作，彼此要相互接納，以後能繁衍後代子孫。 （2）你們不要成為仇敵彼此分離，全部族人都應該同心合一。這是我最後所要吟唱的，我所要說的只有這些了。

  • 出草歌

    啟詞意：孩子起床了，你趕快預備背袋裝你的便當。應詞意：媽媽起床了，我要去工作了，我背了背袋內裝便當，媽媽我要走了。 （背景：根據吟唱者緒述：這個孩子今天起特別早，換他叫母親起床，因為今天他不是要去工作，而是去出草「獵人頭」，所以起特別早，向母親道別，後來他一去就沒有回來。）

  • 織布歌

    第一段詞意：就這樣種苧麻。第二段詞意：就這樣刨麻。第三段詞意：就這樣紡線。第四段詞意：就這樣掛線。第五段詞意：就這樣編織。第六段詞意：就這樣穿衣服。

  • 頭目之歌

    : 第一段詞意：孩子！孩子！我們已經沒有足夠的地方，我們必須往其他的地方去，跟著水走下去，往山上爬過去，找到好的地方生存下去。第二段詞意：泰雅族人無論你去那裡，都要好好的生活，彼此之間不可以爭吵，好讓蒙上帝的憐憫。第三段詞意：泰雅族人，雖然我們曾經受過大災難（指921大地震），若仰望上帝，我不會失望。第四段詞意：各位弟兄姊妹，我們要相互鼓勵，相靠上帝不要軟弱，好讓蒙上帝的憐憫。
由於口簧琴本身的特色與泰雅族人的情有獨鍾，口簧琴及其音樂都和泰雅族的生活有著極為密切的關連。

  • 搗米歌

    詞意：這一首歌是在搗米時所唱的歌謠，通常由三位相互輪唱，歌詞皆屬古文， 已不知其原意。

  • 情歌

    第一段詞意：雖然不曾見過面，但能夠真心相愛，當然因為天天思念，永遠不會忘記恨不得使他的眼睛流下了眼淚。第二段詞意：我本來要在你走的那一刻，要跟隨你後面說再見，但我心中沒有立刻決定，而猶豫不決，不得爬到窗戶，用手帕向你揮揮手，同時他流下眼淚水。

  • 紋面的老人

    第一段詞意：當祖父要被紋面時，是有一定的規範，不論去任何地方，都遵行這個規範，這個規範的內涵，實在無法想像。 第二段詞意：有三位祖父正在倉庫裡烤火，他們很驕傲的談論今天所做的事情。 第三段詞意：祖父在談話之前，會將規範擺在前頭，而換另一位祖父輩說話時，也有他所了解的規範，最後一位祖父輩說完後，他們三位所說的規範是相同的。 第四段 詞意：祖父娶孫媳婦時，他的名字叫獵槍，倉庫裡有床，倍伴著種子、小米、倉庫的底下有牛及水牛，小米的種子如同牛、水牛碩大，將這些合併起來，他有兩個陪伴在這裡。 第五段詞意：從外面將門鎖住,，種子就不斷生成。 第六段詞意：黎明時祖父即前往探望，祖父搭上倉庫的橋時，他就想到泰雅族有一個規範，當他要開倉庫的門時，事先以規範做考量，就是不可以看到不可看到的東西，因此在外面咳了一聲。 第七段詞意：當這孩子長大時，他的父親是獵槍，因他忘了祖父所留存的規範，因此他做所為就毫無忌憚。酒瓶陪伴者他，手在發抖，手腳都在痛風好像香菇那麼大。 第八段大意：祖父痛哭說這孩子是怎麼啦，因此之後，將倉庫改放醃牛肉、小米製成小米酒。 第九段詞意：泰雅族人起來吧！隨著規範行，如此你的倉庫詞能盛滿小米，你才能有好的生活。

  • 祖靈祭歌

    每年到了秋收冬藏之際，泰雅族人都帶著虔誠的心祭拜祖靈，感謝祖靈對後代子孫一年的護佑。
